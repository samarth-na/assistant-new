## Design Context

### Users
Individual power users looking for a reliable, locally-hosted (Ollama) personal AI assistant. The primary context is as a "Desktop Companion"—used alongside other applications to assist with reading, writing, and daily tasks.

### Brand Personality
Calm, utilitarian, focused, and refined. The interface should feel like a high-quality, precision tool, but with an editorial and structured soul rather than a generic chat app.

### Aesthetic Direction
Minimalist, editorial, and structured. The layout will utilize sharp, deliberate lines to compartmentalize content. It embraces a bold, inverted typography scale:
- **Chat/Body Text:** Distinctive Serif (for high-readability, long-form editorial reading)
- **Sidebar/Metadata:** Monospace (to denote system, structure, and utilitarian metadata)
- **Headings/UI Elements:** Clean Sans-serif (for bold, structured architecture)
  
The UI will support four distinct themes to suit any environment: Pure Black, Dark, Cool White, and Warm White.

### Design Principles
1. **Editorial Architecture:** Use deliberate lines and borders to structure space, embracing a print-inspired, grid-heavy layout rather than floating cards.
2. **Typographic Inversion:** Confidently execute the serif-body / mono-sidebar / sans-heading stack with carefully paired custom web fonts.
3. **Intentional Interactions:** Use optimistic UI updates and subtle, purposeful motion (e.g., exponential easing for new messages) rather than bouncy, distracting animations.
4. **No "AI Slop":** Strictly avoid gradient text, side-stripe borders, neon accents, and generic rounded glassmorphism. Build a sharp, serious tool.
